# ***Yank!*** Hardware

3D-printable enclosure for the ***Yank!*** pull switch.

## Files

### 3D Print Files (STL)

| File | Description |
|------|-------------|
| `yank-outer-shell.stl` | Outer shell of the enclosure |
| `yank-internal-top-half.stl` | Internal top half (holds the switch) |
| `yank-internal-bottom-half.stl` | Internal bottom half (holds the microcontroller) |

### 3D Models (Editable)

| File | Description |
|------|-------------|
| `yank-enclosure.f3d` | Full enclosure model (Autodesk Fusion) |

## Print settings

Printed on a Bambu Lab P1S with default settings and supports enabled. No special tuning required.

The battery is sandwiched between the two internal halves.
